## What is this?

This project was created for the Capital One summit application, as apart of their mindsumo challenge.

## How to run

1. Clone the repo
2. Run `yarn install`
3. Run `yarn start`
4. Navigate to [http://localhost:3000](http://localhost:3000) to view it in the browser.
